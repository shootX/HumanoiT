# Taskly CRM

CRM სისტემა პროექტების, ამოცანების, აქტივების და ბიზნეს პროცესების მართვისთვის.

## ტექნოლოგიები

- **Backend:** Laravel, PHP
- **Frontend:** React, Inertia.js, TypeScript, Tailwind CSS
- **ბაზა:** MySQL

## ფუნქციონალი

- **პროექტები** – შექმნა, რედაქტირება, ბიუჯეტი, მილსტოუნები
- **კონტაქტები** – ფიზიკური და იურიდიული პირები, Excel import (ქართული)
- **ამოცანები** – Task stages, კომენტარები, ჩეკლისტები
- **აქტივები** – აქტივების მართვა, კატეგორიები, Excel export/import (ქართული)
- **Bugs** – ბაგების მიკვლევა, სტატუსები
- **Timesheets** – დროის აღრიცხვა
- **ინვოისები** – ინვოისების მართვა
- **კომპანიები და Workspaces** – მრავალკომპანიური რეჟიმი

## ვერსიების ისტორია

### v1.4.0 (2026-02-19)

**აქტივები და ინვოისების ინტეგრაცია**
- Task-ში აქტივების სია – რაოდენობით, ინვოისიდან შექმნილი აქტივები (quantity > 0)
- ამოცანაზე გამოყენებული აქტივების ჩვენება (asset_task pivot)
- ინვოისის რედაქტირება ნებისმიერი სტატუსით
- ინვოისის განახლება – სინქრონიზაცია Asset-ის quantity და task-ის აქტივებთან
- ინვოისის ფორმა – ამოღებულია აქტივის სელექტორი; quantity და rate აბსორბირებენ ათწილადებს
- ინვოისის Index და Show – დამატებულია დაკავშირებული Task
- Assets – მოხსნილია რეზერვის ფუნქცია, დატოვებულია მხოლოდ quantity

**წაშლილი**
- AssetAttachments, AssetWarrantyCase, AssetAttachmentController, AssetWarrantyCaseController

### v1.3.0 (2026-02-14)

**CRM კონტაქტების მოდული**
- ფიზიკური პირი – სახელი, ელფოსტა, ტელეფონი, მისამართი, შენიშვნები
- იურიდიული პირი – შპს სახელი, ბრენდული სახელი, საიდენტიფიკაციო კოდი
- ძებნა, ფილტრი ტიპის მიხედვით, CSV export
- Excel Import – ნიმუში ქართული სათაურებით, სვეტების მიბმა, ქართული სიმბოლოების მხარდაჭერა

### v1.2.x – აქტივების მოდული

- კატეგორიების მართვა – drag & drop რეორდერი
- Excel Export/Import ქართულენოვანი სათაურებით

## ინსტალაცია

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
npm run build
```

## გაშვება

```bash
php artisan serve
npm run dev
```

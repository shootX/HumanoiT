# Taskly CRM

CRM სისტემა პროექტების, ამოცანების, აქტივების და ბიზნეს პროცესების მართვისთვის.

## ტექნოლოგიები

- **Backend:** Laravel, PHP
- **Frontend:** React, Inertia.js, TypeScript, Tailwind CSS
- **ბაზა:** MySQL

## ფუნქციონალი

- **პროექტები** – შექმნა, რედაქტირება, ბიუჯეტი, მილსტოუნები
- **კონტაქტები** – ფიზიკური და იურიდიული პირები, Excel import (ქართული)
- **ამოცანები** – Task stages, კომენტარები, ჩეკლისტები
- **აქტივები** – აქტივების მართვა, კატეგორიები, Excel export/import (ქართული)
- **ტექნიკა (Equipment)** – კონდიციონერები, გენერატორები ფილიალებში, გრაფიკი, QR კოდი, ინვოისიდან ხარჯები
- **ინვოისები** – ინვოისების მართვა, ტექნიკის მოხმარება (ფრეონი, ბენზინი, წმენდა და ა.შ.)
- **კომპანიები და Workspaces** – მრავალკომპანიური რეჟიმი

## ვერსიების ისტორია

### v1.6.0 (2026-02-27)

**ტექნიკის მოდული (Equipment)**
- ტექნიკის CRUD – კონდიციონერები, გენერატორები ფილიალებში (პროექტებში)
- ტექნიკის ტიპები – Equipment Types (კონდიციონერი, გენერატორი და ა.შ.)
- სერვისის ტიპები – Service Types (წმენდა, რემონტი, ფრეონი, ბენზინი)
- გრაფიკი – პერიოდული სერვისის გრაფიკი (ინტერვალი, წინასწარი შეტყობინება)
- QR კოდი – ნამდვილი სკანირებადი QR ტექნიკის გვერდზე
- წაშლა – მხოლოდ company/owner-ს აქვს ტექნიკის წაშლის უფლება

**ინვოისები – ტექნიკის მოხმარება**
- ახალი ტიპი „Equipment consumable“ – ტექნიკის არჩევა + სერვისის ტიპი
- გადახდისას იქმნება ProjectExpense ტექნიკის ფილიალში (არ იქმნება Asset)
- invoice_items – equipment_id, service_type_id ველები

**წაშლილი მოდულები**
- Bugs – ბაგების მიკვლევა
- Timesheets – დროის აღრიცხვა
- რუსული ენა – მოხსნილია მომხმარებლის ენების სიიდან
